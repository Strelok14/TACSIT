# Гибридная схема связи для T.A.C.I.D. (LTE + резерв)

## Цель
Снизить зависимость от LTE за счет резервного канала, не ломая текущую архитектуру.

## Рекомендованная схема
1. Основной канал: LTE (низкая задержка, текущая реализация).
2. Резерв: LoRaWAN для критичных телеметрических сообщений.
3. Перспектива: UWB mesh как R&D трек с отдельным протоколом маршрутизации.

## Почему mesh не включен в прод прямо сейчас
1. DWM3000 поддерживает двусторонний обмен, но полноценная mesh требует:
   - отдельного MAC/роутинга;
   - контроля энергопотребления;
   - тестов устойчивости в условиях multipath/relay.
2. Для боевого внедрения сначала нужен стабильный резерв (LoRa/LTE fallback), затем поэтапный пилот UWB mesh.

## Этапы внедрения
1. Этап 1: LTE + кольцевой буфер на маяке + backlog upload (уже подготовлено в STM32/firmware).
2. Этап 2: Добавить LoRa uplink для пакетов heartbeat/position digest.
3. Этап 3: Пилот UWB mesh на 4-8 узлах и измерение PDR/latency.
4. Этап 4: Решение о расширении mesh на весь полигон.
