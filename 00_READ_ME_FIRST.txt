💡 ВАЖНЫЕ МОМЕНТЫ:

1. API ОТКРЫТ (для тестирования)
   Нет аутентификации по умолчанию. Для production добавьте JWT.

2. БД ИНИЦИАЛИЗИРУЕТСЯ АВТОМАТИЧЕСКИ
   При первом запуске создается SQLite strikeball.db с seed данными.

3. ЛОГИРОВАНИЕ ПОЛНОЕ
   Все запросы логируются в консоль с указанием времени выполнения.

4. SIMULATED МАЯКИ
   BeaconSimulator позволяет тестировать без реального оборудования.

5. REAL-TIME UPDATES
   WebSocket отправляет обновления позиций в real-time всем клиентам.
